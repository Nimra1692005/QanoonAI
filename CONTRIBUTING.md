# Contributing to QanoonAI

Thank you for your interest in contributing to QanoonAI! 

## How to Contribute

### Reporting Bugs
- Use the GitHub Issues page
- Include steps to reproduce, expected behavior, and actual behavior
- Attach screenshots if applicable

### Suggesting Features
- Open a GitHub Issue with the label `enhancement`
- Describe the feature and its benefits for Pakistani users

### Code Contributions

1. **Fork** the repository
2. **Create** a feature branch: `git checkout -b feature/my-feature`
3. **Follow** the code style:
   - Frontend: TypeScript, ESLint rules, Prettier formatting
   - Backend: PEP 8 (Python), type hints required
4. **Write** tests for new features
5. **Commit** with clear messages: `feat: add Urdu OCR support`
6. **Push** and open a **Pull Request**

### Legal Data Contributions
We welcome contributions to `backend/data/legal_data.json`:
- Add more Pakistani court cases
- Add legal terms with Urdu translations
- Improve existing case summaries

### Commit Message Format
```
type(scope): description

Types: feat, fix, docs, style, refactor, test, chore
```

## Code of Conduct
Be respectful and inclusive. This project serves Pakistan's legal community.
